# 泰国清迈 + 曼谷 · 情侣 7 日逐日行程（2026 / 7 / 14 – 7 / 20）

> [!abstract] 行程结构总览
> - **曼谷（前段）** 7/14 周二下午 + 7/15 周三上午 → 约 **1.5 天**，把曼谷"必打卡"放这里（时间更充裕）
> - **清迈（主场）** 7/15 周三夜入住 → 7/16 周四 / 7/17 周五 / 7/18 周六 → **3 个整天 + 首尾**
> - **曼谷（后段）** 7/19 周日下午晚上 + 7/20 周一上午 → 约 **0.75 天**（7/20 只够吃个早餐，无观光时间）
>
> 节奏：自然户外 + 动物 + 美食夜市 + 按摩慢生活，均衡不赶。全程以 **Grab / 出租车 / BTS-MRT-ARL** 为主，清迈近郊用 **包车（红双条 / Grab / 一日司机）**。
