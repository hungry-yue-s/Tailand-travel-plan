# 泰国清迈 + 曼谷 · 情侣 7 日逐日行程（2026 / 7 / 14 – 7 / 20）

> [!abstract] 行程结构总览
> - **曼谷（前段）** 7/14 周二下午 + 7/15 周三上午 → 约 **1.5 天**，把曼谷"必打卡"放这里（时间更充裕）
> - **清迈（主场）** 7/15 周三夜入住 → 7/16 周四 / 7/17 周五 / 7/18 周六 → **3 个整天 + 首尾**
